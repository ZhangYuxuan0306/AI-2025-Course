# ICL 提示策略对比系统 - 研究生人工智能课程大作业

## 项目简介

本项目是研究生人工智能课程的大作业，由第十组完成。项目主题为**ICL（In-Context Learning）提示策略对比系统**，主要研究不同提示策略在大语言模型任务中的性能表现和DSPY功能。

### 项目背景

- **课程**：研究生人工智能课程
- **小组**：第十组
- **主题**：ICL提示策略对比研究
- **目标**：系统性地分析和比较不同提示策略在NLP任务中的效果

## 项目工作内容

### 1. 提示方法任务
**主要工作**：策略对比研究
- **测试策略**：
  - 少样本提示（Few-shot Prompting）
  - 零样本思维链（Zero-shot Chain of Thought）
  - 自一致性（Self-consistency）
- **性能评估**：准确率、响应时间、成本、推理质量
- **任务类型**：文本分类、信息抽取、问答任务

### 2. DSPy模块
**主要工作**：增强功能开发
- **自动提示优化**：基于任务特性的智能提示生成
- **策略搜索**：自动发现最佳提示策略
- **一致性投票**：多路径推理结果聚合
- **交互式推理**：基于DSPy的智能问答

## 技术实现

### 提示方法任务技术栈
- **前端**：Streamlit
- **后端**：Python + OpenAI/DeepSeek API
- **评估**：多维度性能指标
- **可视化**：Plotly图表

### DSPy模块技术栈
- **框架**：DSPy声明式提示优化框架
- **集成**：与基础提示策略的无缝集成
- **优化**：自动提示搜索和模板优化
- **推理**：多路径一致性聚合

## 核心贡献

### 1. 提示策略系统性对比
- 实现了三种主流提示策略的完整对比框架
- 建立了多维度评估指标体系
- 提供了可复现的实验环境

### 2. DSPy增强功能
- 将DSPy框架成功集成到提示工程研究中
- 开发了自动提示优化和搜索功能
- 实现了智能推理和一致性投票机制
